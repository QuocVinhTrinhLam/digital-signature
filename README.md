# DigiSign

A developer-focused web application for digital signature operations using RSA cryptography. 

[GitHub Repository](https://github.com/QuocVinhTrinhLam/digital-signature)

## Features
- **Sign**: Generate RSA-2048 digital signatures for files (SHA-256).
- **Verify**: Check document integrity against signatures.
- **Key Management**: Inspect generated public/private keys.

## Quick Start
```bash
git clone https://github.com/QuocVinhTrinhLam/digital-signature.git
cd digital-signature
npm install
npm run dev
```

## Tech Stack
- Next.js (React, TypeScript)
- Tailwind CSS (Raycast-inspired Design System)
- Node.js Crypto
